btnSprite 視覺化按鍵精靈

版本: 3.1 | 作者: yanchen

功能簡介

btnSprite 是一個視覺化的自動按鍵工具，它能夠自動識別螢幕上的特定圖片，執行按鍵操作，並可在停止時發送LINE通知。

使用說明

  1. 執行 run_key_wizard.bat 或 main.py
  2. 選取「直接點擊模式」(預設開啟)以解決輸入法問題
  3. 點擊「啟動精靈」開始監測螢幕
  4. 程序將自動識別圖片並執行點擊操作
  5. 點擊「停止精靈」可隨時停止

輸入法問題解決方案

預設開啟的「直接點擊模式」可解決中文輸入法問題：

  • 勾選「直接點擊模式」 - 程序將只進行按鈕點擊，不輸入文字
  • 取消勾選 - 程序將嘗試先輸入文字再點擊按鈕

如果還是遇到輸入法問題，請手動將輸入法切換為英文再啟動程序。

設定 LINE 通知

編輯 modules/line_notifier.py 中的兩個變數：

  • 將 LINE_NOTIFY_TOKEN 替換為您的 LINE Notify 令牌
  • 將 USER_ID 替換為您的 LINE 用戶 ID

下載

從 GitHub 下載最新版本：

下載 ZIP

或使用 Git 克隆：

git clone https://github.com/yanchen184/btnsprite-auto.git

必要檔案說明

  • keep.png - 目標圖片
  • btn.png - 按鈕圖片
  • approved.png - 核准圖片 (可選)
  • stop.png - 停止圖片 (可選)

安裝相依套件

pip install -r requirements.txt

疑難排解

若程序自動停止，請檢查:

  • key_wizard.log 檔案中的錯誤信息
  • 確保所有圖片文件都存在且格式正確
  • 檢查 LINE 通知設定是否正確
